# 夜雨试剑 PixiJS 示例

这是一个可直接导入 Mobile Tavern 的横屏武侠即时战斗插件。它使用生成式 2D 背景与角色素材，并由 PixiJS 8 的 WebGL 渲染器实现虚拟摇杆移动、剑式连招、轻功突进、格挡弹反、雨幕、涟漪、粒子特效、镜头反馈和前后台暂停。

敌方会根据距离、玩家挥空次数和弹反倾向执行绕行、逼近、后撤、诱招与追击。普通重击伤害为 24%，强化重击伤害为 27%，配合受击保护可保证玩家至少完整承受三剑。战况气泡由战斗事件驱动，覆盖开场、挥空、受击、弹反与僵持等情境。

战斗使用经典横版单基线：背景和镜头完全固定，双方只进行左右移动；攻击感由动作换图、冲刺、剑光、残影、停顿、火花和涟漪共同提供。

每名角色分别记录原始美术朝向：主角原图朝右、守关人原图朝左，运行时仅在双方越身换位后进行相对翻转。

运行包为每名角色内嵌待机、迈步、挥剑和格挡四张独立人物图。每张动作图都单独生成、抠图和归一化，不再等分裁切动作表，从源头避免相邻姿态越界的剑身碎片串入画面。

走路使用待机与迈步两帧随水平位移稳定切换，不做上下弹跳；挥剑和格挡直接切换完整动作图；受击继续使用稳定站姿配合闪烁和镜头反馈。除轻功保留原有俯身与残影外，普通动作不再旋转、拉伸或摇摆完整人物图。

源码显式引入 `pixi.js/unsafe-eval` 兼容模块，让 PixiJS 改用静态同步实现；宿主 CSP 仍不开放 `unsafe-eval`。

## 构建

在仓库根目录执行：

```powershell
npm run build:example:pixi
```

命令会把 `src/game.ts`、PixiJS 和压缩后的 WebP 美术素材打成单个经典 JavaScript 文件，并生成 `rain-sword-duel.mtplugin`。插件不需要联网，也不依赖宿主预装 PixiJS。图片使用浏览器原生解码，避免在强 CSP 下创建 Worker。
