## [1.5.5] - 2026-06-22

### 🧬 新特性
* **Tauri 移动端底座初始化**：引入并配置了 Tauri 跨平台构建框架，将整体版本升级至 `1.5.5`，奠定了 Android 混合原生 App 的底层架构。
* **移动端主布局与底部导航**：实现了 `MainLayout` 主布局组件，融入了底部大拇指单手可及的导航与全局浮层管理，提供了更顺畅的移动端手势和交互体验。
* **全新预设与正则管理面板**：重构并实现了 `PresetForm` 预设管理组件，支持应用预设、全局正则表达式（Global Regex）与预设正则表达式（Preset Regex）的细粒度添加、修改、导入/导出和批量删除功能。
* **交互式可拖拽看板猫助手**：设计并实现了桌面上可自由拖拽的浮动小猫助手“雪团”（Catbot）及其专属对话界面，接入 `useCatbot` 挂钩，支持在不消耗用户自备额度的情况下为用户提供云端答疑。
* **自动故事总结与年表服务**：引入了全新的 `AutoSummaryService` 服务并与 `useChat` 核心层深度集成，支持在后台自动生成上下文故事大纲与大事年表。
* **富文本渲染增强**：实现了 `FormattedText` 组件，支持安全的 HTML 解析以及自定义 Markdown 复杂表格排版渲染。

### 🔄 修复与优化
* **预设加载与下拉框状态同步**：移除了原下拉菜单中硬编码的“基本预设”占位选项，改为动态计算的 `activeBundleId` 双向绑定，修复了当用户切换预设时，下拉菜单选中状态与实际生效预设不同步的问题。精简并归并了历史预设，仅保留默认的“基本预设”。
* **全局默认聊天背景交互升级**：去除了旧版粗糙的全局默认聊天背景图片 Base64 文本输入框，重构为带状态感知（如 `✨ 已启用自定义背景图片`）的高级功能卡片。按钮支持 `tap-scale` 微动特效，优化了交互体验。
* **控制面板布局与排版调优**：微调了 `SettingsTab` 各功能面板的间距，减少多余空白，优化屏幕纵向空间利用率。新增了 `.custom-scrollbar` 的 `scrollbar-gutter: stable` 规则，有效防止因为滚动条突然显示而导致的整体页面晃动。
* **删除操作视觉降噪**：将预设管理中“批量删除”等敏感操作按钮从原有的高亮红色按钮降级为更柔和的灰色边框按钮，鼠标悬停时才触发红色警示，减少了常规阅读时的界面视觉干扰。

## [1.5.2] - 2026-06-19

### 🧬 新特性
* **解耦微内核架构底座**：在系统底层引入了模块化微内核运行底座 `Kernel`，包含基于 Kahn 算法的依赖拓扑服务装配 `registerServiceBatch` 以及循环依赖熔断，设计了洋葱管道模型中间件机制并提供严格的开发态错误拦截校验。
* **消息总线与故障隔离**：实现了基于优先级排序的高能消息总线 `MessageBus`，新增并行消息发布 `publishParallel` 与超时熔断。支持当非核心服务缺失或发生初始化崩溃时，自动降级为 SafeProxy 代理自愈（SafeProxy 在开发环境下提供强属性访问断言异常，生产环境下支持 No-op 平滑降级与 Promise `await` 链式兼容）。
* **异步取消与一键注销**：服务初始化、消息订阅等异步交互场景支持 AbortSignal 取消。内核暴露一键 destroy 机制，注销时将强制以逆序（拓扑排序逆序）依次触发服务的 destroy 销毁钩子，回收所有的活跃异步任务，杜绝 WebView 内存泄漏。
* **官方微服务及综合测试套件**：将 Database、LLM、Prompt、Telemetry、TableMemory、Script 和 AutoSummary 七大模块全部下沉解耦为独立的微内核服务。同时，新增了覆盖 16 个核心特性的全面自动化测试用例，保证内核的安全运行与可观测性。

## [1.5.1] - 2026-06-17

### 🔄 修复与优化
* **修复**：修复了玩家设定在重命名时，UI 下拉框（SelectTrigger）由于 Radix UI 缓存机制导致名称无法实时刷新的问题。通过在外层包裹容器 `div` 上注入动态 React `key` 属性，强制触发组件重挂载，彻底解决了显示滞后的缓存缺陷。
* **修复**：修复了当人设下拉选择框折叠时，界面由于 Radix 匹配缺陷而 fallback 将技术性的系统生成随机 ID（如 `persona-xxxxxx`）作为当前角色名直接显示在输入框上方的问题。通过直接向 `SelectValue` 的 `children` 传入显式提取的选中人设名称，确保收起时永远展示真实人设名而非系统 ID。
* **优化**：优化了玩家设定管理面板的操作按钮在移动端窄屏下的自适应布局，为“新建”按钮添加了 `shrink-0` 属性，消除并阻止了非对称换行（竖排）的现象，使其与“删除”按钮在所有屏幕宽度下均保持完美、整齐的横向排布。
* **新增**：在项目根目录下为小猫云端大模型增加了 [knowledge.md](file:///d:/projects/Mobile-Tavern/knowledge.md) 专属功能知识库。专门纠正了关于和雪团对话完全不消耗用户自备 Key 额度、自动故事大纲年表总结、玩家人设限制及世界书词条状态的机制说明，用以防止小猫在云端回复用户咨询时产生胡说八道的幻觉问题。
