## [1.5.1] - 2026-06-17

### 🔄 修复与优化
* **修复**：修复了玩家设定在重命名时，UI 下拉框（SelectTrigger）由于 Radix UI 缓存机制导致名称无法实时刷新的问题。通过在外层包裹容器 `div` 上注入动态 React `key` 属性，强制触发组件重挂载，彻底解决了显示滞后的缓存缺陷。
* **修复**：修复了当人设下拉选择框折叠时，界面由于 Radix 匹配缺陷而 fallback 将技术性的系统生成随机 ID（如 `persona-xxxxxx`）作为当前角色名直接显示在输入框上方的问题。通过直接向 `SelectValue` 的 `children` 传入显式提取的选中人设名称，确保收起时永远展示真实人设名而非系统 ID。
* **优化**：优化了玩家设定管理面板的操作按钮在移动端窄屏下的自适应布局，为“新建”按钮添加了 `shrink-0` 属性，消除并阻止了非对称换行（竖排）的现象，使其与“删除”按钮在所有屏幕宽度下均保持完美、整齐的横向排布。
* **新增**：在项目根目录下为小猫云端大模型增加了 [knowledge.md](file:///d:/projects/Mobile-Tavern/knowledge.md) 专属功能知识库。专门纠正了关于和雪团对话完全不消耗用户自备 Key 额度、自动故事大纲年表总结、玩家人设限制及世界书词条状态的机制说明，用以防止小猫在云端回复用户咨询时产生胡说八道的幻觉问题。

## [1.5.0] - 2026-06-17

### 🧬 新特性
* **玩家设定管理**：支持多玩家人设（Persona）的配置、切换、新建与物理删除。
* **历史记录磨砂玻璃美化**：历史对话的卡片样式替换为与角色馆一致的半透明毛玻璃 `.glass-panel`，提高视觉精致感。
