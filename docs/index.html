1 第一步：配置“大脑”（接入 API）

💡 推荐：新手推荐使用 DeepSeek（便宜且足够好用）
  1. 获取密钥：前往 DeepSeek 开放平台 注册账号，创建一个 API Key（以 sk- 开头的一串字符）。
  2. 填入软件：
    • 打开 APP，点击底部最右侧的 【系统设置】。
    • 找到 【接口配置】，接口类型选择 【OpenAI 兼容接口】。
    • Base URL 填入：https://api.deepseek.com/v1
    • API Key 填入你刚才复制的密钥。
  3. 连接测试：点击 【获取模型列表】，选择 deepseek-chat。然后点击 【测试连接】，提示成功即可！

2 第二步：注入“灵魂”（导入角色卡）

💡 生态兼容：完全兼容 SillyTavern 生态 (V2 Spec)
  1. 找卡：前往 Chub.ai 等网站，下载你喜欢的角色卡（必须是 .png 图片格式 或 .json 文本格式）。
  2. 导入：打开 APP，点击底部的 【角色名录】 -> 【导入角色卡】，选择你下载的文件。
  3. 开聊：点击刚导入的角色，进入 【聊天对话】 界面，发送你的第一句话吧！

3 第三步：数据防丢失

🚨 数据备份（极其重要）：

你的所有聊天记录和角色卡都物理保存在手机本地。卸载 APP 或换手机会导致数据彻底清空！请务必在 【系统设置】->【数据管理】 中，定期点击 【导出备份】 并妥善保存文件。

第二部分：详细配置指南（进阶玩家参考）

1. 界面模块导览

客户端主界面底部设有六个核心功能选项卡（Tabs）：

  • 角色名录 (Characters)：管理角色，查看详情，发起新会话。
  • 聊天对话 (Chat)：流式文本渲染、编辑历史消息、切分平行宇宙分支。
  • 会话历史 (History)：会话分支重命名、克隆或删除。
  • 全局词库 (Worldbook)：配置不受角色限制的全局背景设定。
  • 演练场 (Playground)：测试提示词的临时沙盒，不写入本地数据库。
  • 系统设置 (Settings)：配置 API、采样预设、系统提示词以及数据备份。

2. 角色卡导入与管理

完全兼容 SillyTavern 生态规范：

  • 导入角色卡：点击“导入角色卡”导入符合标准的 .png 或 .json。
  • 元数据提取：自动提取角色名称、性格与设定描述、首句问候语、对话示例并存入 IndexedDB。
  • 表情动态映射：自动扫描 AI 回复，通过正则匹配关键词实时切换立绘表情，无规则时自动降级使用主立绘。

3. API 接口与模型配置

A. 使用公网商业 API 中转
  1. 进入“系统设置 -> 接口配置”。
  2. API 接口类型选择“OpenAI 兼容接口”。
  3. Base URL 输入 API 根地址 (如 https://api.deepseek.com/v1)。
  4. API Key 粘贴 Bearer 密钥，点击“获取模型列表”选择模型。
  5. 点击“测试连接”，提示成功即可。
B. 使用本地离线模型 (如 Ollama)
  1. 确保手机与电脑在同一个局域网 (WiFi)。
  2. Base URL 填入本地局域网监听地址 (如 http://192.168.1.100:11434)。
  3. 重要安全限制 (CORS 跨域)：由于 WebView 同源策略，本地 Ollama 启动时需添加环境变量 OLLAMA_ORIGINS=* 并重启 Ollama。

4. 世界设定集配置与使用

通过关键字匹配动态注入设定，极大延长模型记忆：

  • 角色专属词库：绑定在角色卡内的 Lorebook，仅在与该角色对话时触发。
  • 全局共享词库：全局共享设定，所有角色对话均进行统一匹配。
  • 激活逻辑：消息包含关键字时自动将其描述注入上下文，不提及则保持静默。

5. 数据导出备份与覆盖还原

本地数据完全归用户所有：

  • 导出备份：前往“设置 -> 数据管理”，支持 AES-GCM 加密导出和明文导出。文件写入手机 /Download 目录。
  • 导入还原：选择导出的 .json 覆盖还原。注意：导入会覆盖清空当前设备原有全部数据。
