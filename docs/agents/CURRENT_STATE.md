# 当前状态

- 日期：2026-07-21。
- 主线：移动端底座解耦、聊天链路稳定性、Kernel 边界收敛。
- 最近完成：Android 本地角色卡授权改为应用专属的“所有文件访问权限”设置页，允许后自动扫描可访问的共享存储、外置卷和应用共享媒体目录，拒绝后静默停止；系统隔离的 Android 私有区明确跳过。聊天弱网恢复与 GitHub Android 正式签名收紧也已完成。

- 产品决策：不建设独立的规则触发系统、数据派生计算引擎或移动端脚本代码编辑器；变量更新以角色卡 MVU、`mvu_zod`、`UpdateVariable` 和既有沙盒桥接为唯一兼容链路。
