# 当前状态

- 日期：2026-07-22。
- 主线：移动端底座解耦、聊天链路稳定性、Kernel 边界收敛。
- 最近完成：第三方全屏插件第一版与“星穹对决”示例已落地，并修复 Android WebView 在不透明来源沙箱中拒绝父来源 Blob 子资源的问题；当前支持 `.mtplugin` 校验安装、独立存储、全屏强沙箱、自包含资源、存档、退出和方向控制。
- 当前风险：顶层功能使用 `React.lazy`，分包首次进入可能短暂显示加载态；Android 真机冒烟矩阵已完成手动验收，当前未发现明显问题，因此维持现有按需加载策略并继续观察。
- 产品决策：不建设独立的规则触发系统、数据派生计算引擎或移动端脚本代码编辑器；变量更新以角色卡 MVU、`mvu_zod`、`UpdateVariable` 和既有沙盒桥接为唯一兼容链路。
- 下一步：当前没有阻塞发布的已知问题；评估面向非开发者的 AI 插件创作层，以双角色或多 NPC 模板生成气泡交互、游戏状态和 LLM 对话。云端账户、遥测和下载服务暂缓，iOS 不在当前范围内。
