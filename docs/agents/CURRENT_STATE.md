# 当前状态

- 日期：2026-07-22。
- 主线：移动端底座解耦、聊天链路稳定性、Kernel 边界收敛。
- 最近完成：长期记忆升级为事件型片段与可审计记忆包；无标签命中默认不再随机注入旧消息，事件可纠错、失效和保留修订链，重发分支、会话删除与统一备份同步覆盖独立 `memory_fragments` Store。
- 当前风险：顶层功能继续使用 `React.lazy` 按需分包；页签提交已改为 React Transition，快速加载不再闪烁局部加载态，弱性能设备上的长耗时加载仍需继续观察。
- 产品决策：不建设独立的规则触发系统、数据派生计算引擎或移动端脚本代码编辑器；变量更新以角色卡 MVU、`mvu_zod`、`UpdateVariable` 和既有沙盒桥接为唯一兼容链路。
- 下一步：体验验收《星渊终焉》与《夜雨试剑》的横屏触控、性能和战斗节奏；后续评估面向非开发者的 AI 插件创作层，以双角色或多 NPC 模板生成气泡交互、游戏状态和 LLM 对话。云端账户、遥测和下载服务暂缓，iOS 不在当前范围内。
