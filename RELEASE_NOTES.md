# v1.0.0 Release Notes

## 🎉 新增功能 (New Features)
- **核心交互**: 完成了与大语言模型的对话交互界面支持，包含聊天气泡、会话记录、角色卡片等。
- **角色卡系统**: 支持导入、导出和编辑角色卡片，自定义角色设定。
- **配置管理**: 完整的设置面板，支持配置 API Key、选取模型 (Gemini)、切换主题等。
- **历史记录**: 持久化保存聊天记录。

## 🚀 性能与优化 (Enhancements)
- **UI 优化**: 改进了深色/浅色主题的视觉体验与响应式布局。
- **本地存储**: 采用本地存储机制保障用户数据的隐私与安全。

## 🐛 问题修复 (Bug Fixes)
- 初始稳定版本发布，确保核心对话链路顺畅。
